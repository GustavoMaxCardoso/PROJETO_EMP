# 🏰 PROJETO_EMP — Automação de Eventos com Python

> Automação modular em Python com detecção visual, controle de janelas e tratamento de erros para execução autônoma de eventos repetitivos.

---

## 📌 Sobre o Projeto

O PROJETO_EMP é um sistema de automação desenvolvido em Python para executar eventos de forma autônoma, sem interação manual. O projeto nasceu da necessidade de eliminar tarefas repetitivas e de alto volume, aplicando conceitos de **visão computacional**, **controle de interface** e **arquitetura modular**.

Cada evento é tratado como um módulo independente, com múltiplas versões evoluindo progressivamente conforme novos casos de uso são identificados.

---

## ⚙️ Funcionalidades

- 🔍 **Detecção visual** — identificação de blur, obstruções e estado da tela em tempo real
- 🪟 **Controle de janelas** — gerenciamento e foco automático de janelas e pop-ups
- 🔁 **Múltiplas versões** — cada módulo possui versões incrementais com melhorias progressivas
- 🛡️ **Tratamento de erros** — sistema robusto de captura e recuperação de falhas
- 🧩 **Arquitetura modular** — cada evento é um módulo isolado e independente
